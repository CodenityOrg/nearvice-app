can be filtered according to different added characteristics.

## 🚀 Getting Started

```
npm install
npm run dev
npm run build
```

The `npm run dev` command execute for default in port `8080`, change in `vue.config.js` file if you has this port on use.

### Lints and fixes files
```
npm run lint
```
